热烈祝贺闽北谢氏成立20周年

本会动态 \ 2014-8-29

2014年8月24日，闽北谢氏联谊会20周年庆典暨五届一次代表大会在将乐县天源国际大酒店隆重召开。到会祝贺的有福建省谢氏委员会和谢氏厦门、泉州、龙岩、南平、福州、三明等分会，会员代表近200人。大会产生了新一届理事会，谢兴旺当选会长。
